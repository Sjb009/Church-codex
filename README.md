# Church Codex

Starter project for a church admin web app with three core modules:

- Announcements Board
- Prayer Request App
- Attendance Tracker

## Stack

- Next.js
- TypeScript
- Prisma
- SQL Server

## First steps

1. Copy `.env.example` to `.env`.
2. Update `DATABASE_URL` for your local SQL Server instance.
3. Install dependencies with `npm install`.
4. Generate the Prisma client with `npm run prisma:generate`.
5. Start the app with `npm run dev`.
